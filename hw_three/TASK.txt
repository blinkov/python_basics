Реализовать на выбор:
- блог
- форум
- чат (рекомендуется только есть знания JavaScript)
Выбор используемых технонологий и библиотек остается за вами, нельзя использовать только полностью готовые решения.

Требования:
- редактируемое через админку название (title)
- регистрация с активацией по почте (можно использовать готовую)
- авторизация
- поддержка RSS
- постраничное отображение
- даты отправки сообщений

Для блогов:
- посты
- черновики
- комментарии для зарегистрированных и анонимных пользователей

Для форумов:
- разделы
- обсуждения
- посты

Для чата:
- сообщения в реальном времени (задержка не более 10 секунд)


Дополнительный функционал:
- теги
- смайлики
- авторизация по OpenID
- авторизация по соц. сетям
